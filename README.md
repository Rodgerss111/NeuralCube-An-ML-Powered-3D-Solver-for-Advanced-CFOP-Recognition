# NeuralCube-An-ML-Powered-3D-Solver-for-Advanced-CFOP-Recognition
  An interactive web application designed to guide users through the advance stages of solving a 3x3 Rubix's Cube using the Fridrich Method (CFOP). The app focuses on the technical implementation of F2L (First Two Layers), OLL (Orientation of the Last Layer), and PLL (Permutation of the Last Layer), assuming the initial "Cross" is already complete. 
  
  Most existing AI Rubik's cube solvers (DeepCubeA, Kociemba, AlphaZero-style) treat the cube as a pure optimization problem — find the shortest path to solved, full stop. NeuralCube is different because it mirrors how a human actually learns: a neural network attempts first from pattern recognition, falls back to structured case logic when it fails, and only reaches for guaranteed algorithms as a last resort — making the failure modes transparent and the reasoning interpretable at every layer. Specifically, the edge-AI angle makes this genuinely novel: a lightweight, exportable TensorFlow.js model that runs inference at the browser edge with no server, combined with a graceful degradation pipeline, is something the academic solvers never needed to care about and the toy demos never bothered to build properly.
